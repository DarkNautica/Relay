How It Works

Three steps to real-time. That's it.

1

Run the Server

Single binary or Docker container. docker run relayhq/relay and you're live on port 6001.

2

Connect from JavaScript

Install @relayhq/relay-js and subscribe to channels. Events arrive in real-time over WebSocket.

3

Publish from Your Backend

Use any server SDK — Laravel, Node, Rails, or Django — to publish events via a simple HTTP call.
